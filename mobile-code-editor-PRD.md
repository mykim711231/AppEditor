# 모바일 코드 에디터 PWA — 프로젝트 기획서 (PRD)

> 버전: v1.0 | 작성일: 2026-06-20 | 상태: 확정

---

## 1. 프로젝트 개요

### 목적
앱스토어·개발자 계정 없이 아이폰 8 및 삼성 갤럭시 태블릿(안드로이드)에서  
코드 작성, 오타 방지, 괄호 확인, 멀티 AI 문의를 하나의 앱에서 사용할 수 있는  
PWA(Progressive Web App) 코드 에디터를 개발한다.

### 타겟 기기
| 기기 | OS | 브라우저 |
|------|-----|---------|
| 아이폰 8 (4.7인치) | iOS | Safari |
| 삼성 갤럭시 태블릿 | Android | Chrome / Samsung Internet |

### 핵심 원칙
- 앱스토어 등록 불필요
- 개발자 계정 불필요
- API 키 불필요 (AI 연동)
- 완전 무료 운영 가능
- 한국어 UI 전용

### 2-4. 멀티 AI 웹챗 연동

#### 동작 방식
API 키 없이 각 AI 웹사이트로 코드+질문을 전달하는 방식.  
사용자가 각 AI 사이트에 브라우저로 1회 로그인해두면 이후 자동으로 해당 사이트가 열림.

```
코드 에디터에서 코드 선택
        ↓
질문 텍스트 입력
        ↓
질문 범위 선택 (선택한 코드만 / 전체 코드)
        ↓
AI 버튼 탭
        ↓
코드 + 질문 클립보드 자동 복사
        ↓
해당 AI 웹사이트 새 탭으로 열림
        ↓
붙여넣기 → 질문 완료 ✅
```

#### 기본 제공 AI 목록

| AI | URL | 무료 여부 |
|----|-----|---------|
| Claude | claude.ai | 무료 계정 |
| ChatGPT | chatgpt.com | 무료 계정 |
| Gemini | gemini.google.com | 무료 계정 |
| Qwen | chat.qwen.ai | 무료 |
| Kimi | kimi.moonshot.cn | 무료 |
| DeepSeek | chat.deepseek.com | 무료 |
| Grok | grok.com | X 계정 필요 |
| Perplexity | perplexity.ai | 무료 |
| Le Chat (Mistral) | chat.mistral.ai | 무료 |

#### AI 관리 기능

| 기능 | 상세 | 상태 |
|------|------|------|
| 신규 AI 추가 | 이름 + URL + 아이콘 이모지 입력 | ✅ 확정 |
| AI 버튼 순서 변경 | 드래그 앤 드롭 | ✅ 확정 |
| AI 버튼 삭제 | 버튼 길게 누르기 | ✅ 확정 |
| AI 목록 저장 | localStorage 영구 저장 | ✅ 확정 |

#### AI 질문 범위 선택

```
질문 범위:  ● 선택한 코드만   ○ 전체 코드
```

### 2-6. 메뉴 숨김 기능 (화면 공간 확보)

#### 숨김 제어 방식

| 제어 방법 | 동작 | 기기 |
|---------|------|------|
| ≡ 햄버거 버튼 | 사이드 메뉴 토글 | 공통 |
| 👁 버튼 | 파일바·AI바 동시 숨김·표시 | 공통 |
| 스와이프 위 | AI 패널 표시 | 터치 기기 |
| 스와이프 아래 | AI 패널 숨김 | 터치 기기 |
| Cmd+B | 사이드바 토글 | 외부 키보드 |
| Esc | 전체 집중 모드 | 외부 키보드 |

#### 화면 모드

```
【기본 모드】
┌─────────────────┐
│ ≡  파일명.js  👁│  ← 상단 탭바
├─────────────────┤
│ 📁폴더▼ 📄파일▼ │  ← 파일바
├─────────────────┤
│   코드 편집기   │
├─────────────────┤
│Claude GPT Gemini│  ← AI 버튼
└─────────────────┘

【집중 편집 모드 (👁 탭)】
┌─────────────────┐
│ ≡  파일명.js  👁│  ← 탭바만 유지
├─────────────────┤
│                 │
│   코드 편집기   │  ← 최대 화면
│   (전체 공간)   │
│                 │
└─────────────────┘

【AI 패널 모드 (스와이프 위)】
┌─────────────────┐
│ ≡  파일명.js  👁│
├─────────────────┤
│   코드 편집기   │  ← 축소
├─────────────────┤
│ 질문: [입력창  ]│
│ ●선택코드 ○전체 │
│Claude GPT Gemini│
│Qwen  Kimi  [+] │
```

## 3. 최종 기술 스택

```
프론트엔드 코어
├── React 18                    UI 컴포넌트
├── Tailwind CSS                반응형 스타일 (한국어 UI)
└── Vite                        빌드 도구

코드 편집기
├── CodeMirror 6                신택스·괄호·자동완성
├── @codemirror/language-data   80개+ 언어 자동 감지
└── @codemirror/theme-*         다크·라이트 테마

파일·폴더 관리
├── IndexedDB                   브라우저 내부 저장 (기본)
├── File System Access API      로컬 폴더 (갤럭시·PC)
├── Google Drive API            구글 드라이브 저장
├── Google OAuth 2.0            구글 로그인 인증
└── dnd-kit                     파일·폴더·AI버튼 순서 변경

AI 연동
├── Clipboard API               코드+질문 자동 복사
├── window.open()               AI 웹챗 새 탭 열기
└── localStorage                AI 목록·순서·신규 추가 저장

부가 기능
├── localStorage                스니펫·폰트·테마 설정 저장
└── IndexedDB                   AI 질문 히스토리 저장

메뉴 숨김
├── React useState              숨김·펼침 상태 관리
├── CSS transition              부드러운 애니메이션
├── Touch Events (스와이프)      스와이프 메뉴 제어
└── KeyboardEvent               단축키 메뉴 토글

PWA
├── manifest.json               홈 화면 설치·아이콘
├── Service Worker              오프라인 캐시
└── Virtual Keyboard API        키보드 레이아웃 대응

모바일 최적화
├── Pointer Events API          터치·마우스·S Pen 통합
├── CSS Media Query             기기별 레이아웃 분기
└── viewport meta tag           모바일 줌 방지

호스팅
└── Vercel                      무료 HTTPS 자동 배포
```

## 5. 제약 사항 및 주의 사항

| 항목 | 내용 |
|------|------|
| 로컬 폴더 저장 | 아이폰 Safari 미지원 → IndexedDB 또는 Google Drive 사용 |
